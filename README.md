## Requirements

 - [Node v14.0+](https://nodejs.org/en/download/current/)

## Getting Started s
```
Node Version 14+

Install dependencies:

```bash
npm install
```

Set environment variables in `.env`. Use `.env.example` for an example of the key names.

### Running the API server
```bash
npm run dev
```

## Running in Production

```bash
npm run start
```
